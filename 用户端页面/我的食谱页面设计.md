```
┌─────────────────────────────────────────────────────────────────────────────────────────────┐

│  ┌───────────────────────────┬─────────────────────────────────────────────────────────────┐ │
│  │  🤦‍♂️ [头像=个人中心]       │ 【我的食谱】                                      🎨 筛选  │ │
```

## 核心设计点
1. **标签云筛选**：按菜品种类/场景/难度快速筛选
2. **食谱卡片**：展示名称、卡路里、难度（🔴中等/🟡困难/🟢超简单）
3. **快捷操作**：创建食谱、分享食谱、批量管理
4. **网格布局**：3列卡片展示，适配桌面端大屏

## 交互逻辑
- 点击标签云筛选对应类型食谱
- 点击卡片进入食谱详情页
- 支持编辑/删除/收藏/分享食谱
- 支持按时间/名称/卡路里排序
