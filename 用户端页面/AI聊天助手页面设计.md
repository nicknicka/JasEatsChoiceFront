# 🤖 AI聊天助手页面设计
## 统一基础框架
```
┌─────────────────────────────────────────────────────────────────────────────────────────────┐
│ 【佳食宜选】                          [🔍 搜索框(居中)]                      👤/🏪 | 用户名   │
├─────────────────────────────────────────────────────────────────────────────────────────────┤
│  ┌───────────────────────────┬─────────────────────────────────────────────────────────────┐ │
│  │  🤦‍♂️ [头像=个人中心]       │ 【AI饮食助手】                                    📚 历史  │ │
│  │  📌 (我的推荐)            │ ┌─────────────────────────────────────────────────────┐       │ │
│  │ 🏪 (商家查找)             | |                                                       |     |  |
│  │  📅 (今日食谱)            │ │  🎯 快捷功能：生成减脂食谱 | 食材搭配建议 | 卡路里查询    │       │ │
│  │  📊 (卡路里统计)          │ │  ┌─────────────────────────────────────────────┐           │ │
│  │  📚 (我的食谱)            │ │  │ 🤖 AI: 您好！请问有什么饮食问题需要帮助？        │ 对话区   │ │
│  │  💬 (消息中心)            │ │  │ 💬 我: 帮我生成300kcal的减脂午餐                │           │ │
│  │  🤖 (AI聊天助手=当前页)    │ │  │ 🤖 AI: 推荐...总300kcal                     │           │ │
│  │  ⚙️ (设置-底部固定)       │ │  └─────────────────────────────────────────────┘           │ │
│  └───────────────────────────┴─────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────────────────────────────────┘
```

## 核心设计点
1. **快捷功能入口**：预设高频使用的AI功能
2. **对话界面**：气泡式消息展示，支持文本/图片/语音输入
3. **历史记录**：支持查看历史对话，保存AI生成的内容
4. **交互反馈**：AI生成内容可直接保存为食谱或分享

## 交互逻辑
- 点击快捷功能快速发起提问
- 支持上传食材图片识别
- 语音输入自动转为文字
- AI生成的食谱可一键保存
- 支持清除对话记录
