# 📢 系统通知详情页面设计
## 统一基础框架
```
┌─────────────────────────────────────────────────────────────────────────────────────────────┐
│ 【佳食宜选】                          [🔍 搜索框(居中)]                      👤/🏪 | 用户名  │
├─────────────────────────────────────────────────────────────────────────────────────────────┤
│  ┌───────────────────────────┬─────────────────────────────────────────────────────────────┐ │
│  │  🤦‍♂️ [头像=个人中心]       │ 【系统通知详情】                                  ↩ 返回   │ │
│  │  📌 (我的推荐)            │ ┌─────────────────────────────────────────────────────┐       │ │
│  │ 🏪 (商家查找)             | |                                                       |     |  |
│  │  📅 (今日食谱)            │ │  📢 立秋贴秋膘特惠活动已开始！                        │       │ │

│  └───────────────────────────┴─────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────────────────────────────────┘
```
## 核心设计点
1. **完整内容展示**：清晰呈现通知的标题、时间、详细内容及活动福利
2. **便捷操作入口**：提供"查看活动详情"、"领取优惠券"等直接操作按钮
3. **关联功能**：支持查看更多通知的快捷入口
4. **统一框架**：保持系统一致的左侧导航栏与顶部搜索功能
5. **返回机制**：右上角设置返回按钮，便捷返回消息中心
