# 💸 消费记录页面设计
## 统一基础框架
```
┌─────────────────────────────────────────────────────────────────────────────────────────────┐
│ 【佳食宜选】                          [🔍 搜索框(居中)]                      👤/🏪 | 用户名   │
├─────────────────────────────────────────────────────────────────────────────────────────────┤
│  ┌───────────────────────────┬─────────────────────────────────────────────────────────────┐ │
│  │  🤦‍♂️ [头像=个人中心]       │ 【消费记录】                                  🔍 筛选   │ │
│  │  📌 (我的推荐)            │ ┌─────────────────────────────────────────────────────┐       │ │
│  │ 🏪 (商家查找)             | |                                                       |     |  |
│  │  📅 (今日食谱)            │ │  交易时间     交易类型  金额    状态      备注          │       │ │
│  │  📊 (卡路里统计)          │ │  2024-11-20   订单支付  -89元   已完成   凉面套餐+西瓜汁    │       │ │
│  │  📚 (我的食谱)            │ │  2024-11-19   充值      +500元  已完成   支付宝充值        │       │ │
│  │  💬 (消息中心)            │ │  2024-11-18   订单退款  +56元    已完成   取消订单退款      │       │ │
│  │  🤖 (AI聊天助手)          │ │  2024-11-17   订单支付  -67元   已完成   青木瓜沙拉        │       │ │
│  │  ⚙️ (设置-底部固定)       │ └─────────────────────────────────────────────────────┘       │ │
│  │                          │  [上一页] 1/3 [下一页] | 每页10条 | 筛选：[全部/支付/充值/退款] │       │ │
│  └───────────────────────────┴─────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────────────────────────────────┘
```
## 核心设计点
1. **交易记录展示**：采用表格布局，清晰显示交易核心信息（时间、类型、金额、状态、备注）
2. **金额标识**：支出用"-"红体、收入用"+"绿体，视觉区分明显
3. **交易类型**：支持筛选支付/充值/退款等不同交易类型
4. **备注信息**：显示具体交易内容，便于用户回忆交易场景
5. **分页功能**：支持分页展示，提升大数据量下的浏览体验
